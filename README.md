### 📝 Real-Time Collaborative Document Editor

## 📌 Project Overview

This project is a **real-time collaborative document editor** that allows multiple users to edit the same document simultaneously. Any changes made by one user are instantly reflected for all other connected users without requiring a page refresh.

The application demonstrates **real-time synchronization**, **frontend-backend integration**, and **data persistence** using modern web technologies.

---

## 🚀 Features

* Real-time collaborative editing
* Instant synchronization across multiple users
* No page reload required
* Dynamic and responsive UI
* Automatic content updates
* Backend integration for data handling

---

## 🛠️ Technologies Used

### Frontend

* React.js
* JavaScript
* HTML5
* CSS3

### Backend

* Node.js
* Express.js
* Socket.IO

### Database

* MongoDB (optional / can be replaced with in-memory storage)

---

## ⚙️ How It Works

1. The user opens the document editor in the browser
2. The frontend connects to the backend server using Socket.IO
3. When a user edits the document, the updated content is sent to the server
4. The server broadcasts the updated content to all connected clients
5. All users see the changes in real time
